# Reference Protocol

Use this when the user shares screenshots, URLs, or style references but does not want a template-like result.

## Rule

Never copy a reference wholesale.
Split each reference into borrowable dimensions, then recombine them through the chosen city, its food, and its soundscape.

## Reference Source Classification (run first)

Before extracting any dimensions, classify the reference:

| Type | Examples | Risk |
|------|---------|------|
| **Social / community platform** | Instagram, X, Threads, Behance, Dribbble, Pinterest, LinkedIn | High — surface aesthetic will overwrite urban/cultural language |
| **Content aggregator / feed** | Medium, Substack, news sites, Reddit-style layouts | High — feed rhythm replaces district rhythm |
| **Brand / campaign site** | Product launches, agency showreels, fashion brands | Medium — borrow framing and restraint only |
| **Editorial / publication** | Magazine layouts, long-form journalism | Low — rhythm and density translate well |
| **Film / cultural institution** | Museum sites, film festival pages, art-house brands | Low — already in urban/cultural territory |

**If the reference is High risk:** State this explicitly. Extract only rhythm, density, and navigation posture. Discard the surface aesthetic entirely. Rewrite everything through the urban/cultural language. Do not let the reference's visual identity survive into the output.

**If the reference conflicts with the chosen city or culture:** Tell the user. Ask whether to keep the city/culture or choose a more compatible reference approach. Do not silently blend the two.

## Allowed Borrowing Dimensions

- rhythm
- density
- typography attitude
- image treatment
- materiality
- framing
- interaction restraint
- navigation posture

## Forbidden Borrowing

- copying the full section order
- recreating the same hero composition
- reusing the same color combination without urban justification
- cloning the same card system across the whole page
- borrowing a reference's grid skeleton as the project's fallback layout

## Safe Reference Method

When there are multiple references, assign each one a role:

- Reference A: rhythm and pacing
- Reference B: material and surface treatment
- Reference C: typography and editorial attitude

Then rewrite all three through the chosen city, artist, and soundscape.

## Output Format

Record this in `decisions.md`:

- `Reference A contributes:`
- `Reference B contributes:`
- `Reference C contributes:`
- `What will not be copied:`
